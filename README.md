# Neural_Network_ML

```
To run the application:
Requirements:
    Python3.5 and higher.
    The provided requirements file has the libraries needed to run the application.
    To install :
        pip install -r requirements.txt

Running in order as follow:
    Run:
        1) python3 prep_data.py
            Explain: This preps the training and test data. and create files for them.
        2) python3 nn_main.py
            Explain: This runs the neural network for different layers:
                This will produce different files for graphs and table of the mse
                for training and testing data.
```
